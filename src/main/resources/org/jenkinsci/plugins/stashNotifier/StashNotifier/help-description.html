A TokenMacro compatible string to use as the notification sent to Stash. Activated when "use job description" parameter is unchecked. As part of TokenMacro support, this plugin defines a new macro, ${BUILD_RESULT}, that resolves to the current build's result (i.e. SUCCESSFUL, FAILED, etc).
If left blank, default description used is:

  • Build ${BUILD_DISPLAY_NAME} on ${NODE_NAME}: ${BUILD_RESULT}
which gets rendered as Build #123 on master: SUCCESSFUL
